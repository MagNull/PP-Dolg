# Работа в кампусе

Учебный проект на FastAPI для поиска студенческих вакансий.

- **Студент** может смотреть вакансии, фильтровать их, откликаться и видеть статус своего отклика.
- **Работодатель** может зарегистрироваться, создать профиль компании, добавлять вакансии через личный кабинет, указывать навыки и управлять откликами кнопками `Принять` / `Отклонить` / `Просмотрено`.

## Что используется

- FastAPI
- SQLAlchemy
- SQLite
- Bootstrap 5
- pytest + httpx

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:8000
```

## Тесты

```bash
source .venv/bin/activate
pytest tests/ -v --tb=short
```

## Что есть в проекте

- регистрация студента и работодателя;
- автоматическое создание профиля работодателя при регистрации;
- каталог вакансий с быстрыми фильтрами;
- просмотр вакансии и отклик с необязательным сопроводительным письмом;
- отображение статуса отклика у студента;
- создание вакансии с навыками через поиск и теги;
- просмотр откликов работодателем и смена статусов кнопками;
- схема БД в `docs/db_schema.png`.

## Структура проекта

```text
app/                 backend на FastAPI
static/              HTML, CSS, JS
tests/               тесты
docs/db_schema.png   схема базы данных
```
